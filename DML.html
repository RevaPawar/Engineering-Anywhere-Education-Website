1)Select Command


The SELECT keyword is a fundamental component of the SQL language.

It is used to retrieve data from one or more database tables. With the SELECT keyword,

you can specify the columns you want to retrieve, apply filtering conditions using

The SELECT keyword is versatile and essential for data retrieval and analysis in SQL.





2)Insert Command


The INSERT keyword in SQL is used to add new rows of data into a table.

It allows you to specify the table name and column names for insertion.Values or

data to be inserted can be provided explicitly or from another query result using the

SELECT statement. INSERT enables you to add data and expand the contents of your database tables.





3)Update Command


The UPDATE keyword is used in SQL to modify existing data in a table.

It allows you to specify the table to be updated, set new values for specific columns,

and apply conditions using the WHERE clause to selectively update rows that meet

certain criteria. The UPDATE statement helps in altering data within a database.





4)Delete Command


The DELETE keyword in SQL is used to remove one or more rows from a table

It allows you to specify a condition using the WHERE clause to determine which

rows should be deleted. Deleting data using the DELETE statement permanently

removes it from the table and cannot be easily recovered.
