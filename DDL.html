1) Create a table


To create a table in MySQL, use the "CREATE TABLE" statement

followed by the table name and column definitions.

Specify the column names, data types, and any constraints like primary key or foreign key.

Additional settings like default values or character sets can be included as well.






2) Alter a table


ALTER TABLE is a SQL command used to modify an existing table's structure.

It can add or drop columns, modify column data types or constraints, rename columns,

and add or remove indexes. ALTER TABLE allows for flexible modification

of table structure without losing existing data.






3) Drop a table


The DROP TABLE command in SQL is used to delete an entire table and

its data from the database. It permanently removes the table,

ncluding all associated columns, indexes, and constraints.This command should

be used with caution as it irreversibly deletes the table and its contents.






4) Truncate a table


TRUNCATE TABLE is a SQL command used to delete all rows from a table,

effectively removing all data while keeping the table structure intact.

Unlike the DELETE command, TRUNCATE TABLE is faster as it does

not generate individual row-level delete operations and cannot be rolled back.


Point to be Remembered : SQL is a not a Case-Sensitive Language.

Some Important Syntax



  • CREATE DATABASE database_name;

  • CREATE TABLE table_name( column1 datatype, column2 datatype, ..... columnN datatype );

  • DROP TABLE table_name;

  • DROP DATABASE database_name;

  • ALTER TABLE table_name ADD column_name column_definition;

  • truncate table my_table; select * from my_table;

  • RENAME TABLE table_name TO table_name_new;
